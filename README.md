# 백준 JS 자동 IO 포멧팅

자바스크립트로 백준 문제 푸실때 IO 를 직접 입력하실 필요 없습니다.
자동으로 세팅해드리겠습니다.

## Install

### npm

```sh
npm install --save-dev @jaeilnet/backjoon_cli
```

#### yarn

```sh
yarn add --dev @jaeilnet/backjoon_cli
```

#### pnpm

```sh
pnpm add --save-dev @jaeilnet/backjoon_cli
```

## use

```sh
npx backjoon-cli start
```
